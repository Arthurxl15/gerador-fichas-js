# Gerador de Fichas JS

Repositório para fichas (templates PDF) e livros em PDF, com app web em JavaScript.

## Como Usar
1. Instale Node.js de [nodejs.org](https://nodejs.org).
2. Rode `npm install` para instalar dependências.
3. Execute `npm start`.
4. Acesse `http://localhost:3000`.

## Estrutura
- `fichas/`: Templates PDF de fichas.
- `livros/`: Livros em PDF para extração.
- `uploads/`: Arquivos temporários.
- `outputs/`: Fichas geradas.
